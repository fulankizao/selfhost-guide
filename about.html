📌 About Self-Host Guide

Self-Host Guide is a hands-on Homelab tutorial site for overseas Chinese developers. Topics include AI Agent deployment, Linux server operations, web service deployment, and more. All tutorials are written based on real hands-on experience, not theoretical regurgitation.

👤 About the Author — Frankie

I'm Frankie, a full-stack developer with years of experience in Linux server and WSL environment operations. I started self-hosting in 2025 and have deployed and maintained the following services:

  • Hermes Agent — Full AI agent framework setup, QQ Bot integration, and production environment operations
  • Dashboard + Gateway — systemd-based auto-start service architecture
  • FastAPI Web Services — End-to-end workflow from development to Nginx reverse proxy deployment
  • WSL2 Production Environment — Disk management, proxy tunneling, and service auto-start optimization

My principle: Every article is based on real problems I've encountered and solved. I don't publish tutorials I haven't verified myself.

🎯 Why This Site Exists

Systematic self-hosting tutorials are scarce in the Chinese-language internet — most content is scattered across forums and blogs, and much of it is outdated. This site aims to provide actionable guides you can follow and get working, helping more Chinese-speaking developers master self-hosting technologies.
